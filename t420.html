the thinkpad t420 in 2023

Hey, this is my first post on this blog, so what is a better way to celebrate it than review my daily driver, the ThinkPad T420!

Specs

The specs when I got it were like this:

Specs
Processor I5 2450M
Memory 4GB DDR3
Drive 500GB HDD

After I upgraded it to this state:

Specs
Processor I5 2450M
Memory 8GB DDR3
Drive(s) 256GB SSD, 500GB HDD (more on it later)

My first upgrade was upgrading the RAM to 8 gigabytes and after upgrading the SSD and getting an ultrabay caddy to use the 500GB drive for data.

Build quality

The build quality is insane, one time I was on vacation I accidentaly bumped the thing into a wall and it dented the wall and left small cosmetic damage. Other than that, the keyboard is the best keyboard on anything. I’m fortunate that I got the NMB keyboard which is the best one. Funny enough I use the Keychron K4 with my desktop and I’m faster with the Thinkpad keyboard. The keycaps are a bit oily now due to excessive typing, but they weren’t when I got it. The trackpad is okay, but I don’t use it very much since I use the trackpoint which is very useful. The screen is 768p and kinda meh, but I will upgrade it to either a 1080p IPS or a 900p IPS panel. The speakers work, but they aren’t great, but I don’t use them, because I wear headphones all the time. Also, the microphone is kinda decent, but as I said I use headphones all the time, which have a microphone.

Performance

The performance is fine, it is Sandy Bridge in 2023 so don’t expect something special, but with Linux I can’t find many performance issues. Since I use qutebrowser, which is lightweight, web browsing isn’t a problem. My setup is Arch Linux with DWM, which still works and works well for my workflow which consists of watching videos and occasional “programming” (ricing my desktop). Screen recording simple videos in 720p still works and even watching videos in 720p works fine (1080p is pointless for a 768p screen). It also works with firefox, but qutebrowser is better for productivity and eats fewer resources (which is weird since qutebrowser is written in Python). Also, a thing to note, it can drive two monitors, but the internal one has to be disabled. As for the graphics, my model has the Intel HD 3000 internal graphics, but some models have the NVIDIA Quadro NVS4200M graphics for better or for worse.

Conclusion

If you want to buy it and this would be your only machine, don’t buy it. The Thinkpad T480 is a much better choice nowadays since it has a quad-core 8th Gen i5 which destroys the weak dual-core i5 from 2011. But if you find it under 50 Euros or American Freedom Eagles, then fine. It is a great deal. In $CURRENT_YEAR it can still work as a word processor or a programming machine. It can even work as a headless server when docked in a docking station (which is dirt cheap, i got mine for 7 euros shipped). So, is it worth it to buy? From what I said, decide yourself.
